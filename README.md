A full-stack Patreon-inspired crowdfunding platform built with Next.js, React, MongoDB, and NextAuth.
This project enables creators to receive support and donations from their audience through a modern, secure, and responsive interface.

Features
User authentication with NextAuth
Creator profiles
Support creators with donations
Secure payment integration
Responsive UI
MongoDB database integration
Tech Stack
Frontend: Next.js, React, Tailwind CSS
Backend: Next.js API Routes
Authentication: NextAuth
Database: MongoDB
